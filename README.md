# Speech Multi-Classifier

## Introduction
The Speech Multi-Classifier is a machine learning project designed to classify audio inputs into different categories. This project aims to provide an efficient and scalable solution for speech classification tasks.

## Features
- Multi-class classification of audio inputs.
- Support for various audio formats.
- Efficient model training and evaluation.

## Installation
To install the required dependencies, please run:
```bash
pip install -r requirements.txt
```

## Usage
To use the Speech Multi-Classifier, follow these steps:

### Training
```bash
python train.py --data <path_to_data>
```

### Inference
```bash
python infer.py --audio <path_to_audio>
```
